# Goodreads Obsidian Base Readlist

[Tampermonkey](https://www.tampermonkey.net/) script to Copy Goodreads data to Markdown for Obsidian base.

[Script](https://raw.githubusercontent.com/siddacool/automation-scripts/gh-pages/packages/goodreads-obsidian-base-readlist/goodreads-obsidian-base-readlist.user.js)

## Features

Copy Goodreads data to Markdown for Obsidian base.

## How to install (Chrome)

1. The script is created for [Tampermonkey](https://www.tampermonkey.net/). So first you need to install Tampermonkey [chrome extenstion](https://chromewebstore.google.com/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo).

2. Click on Tampermonkey extension and select 'Dashboard' from the menu

3. Click on "Utilities" tab, scroll down to "Import from URL" textbox.

4. copy [Script](https://raw.githubusercontent.com/siddacool/automation-scripts/gh-pages/packages/goodreads-obsidian-base-readlist/goodreads-obsidian-base-readlist.user.js) and paste it in "Import from URL" textbox.

5. Click install
